# 🎮 Max星际先锋：本地运行与战机皮肤自定义指南

本指南将帮助你在本地电脑（VSCode 运行环境）中，替换和自定义游戏中的主角战机和敌方战机的 PNG 贴图。

---

## 1. ⚙️ 本地部署步骤

首先，通过 **AI Studio Settings** 菜单将项目导出为 **ZIP** 压缩包，或者直接同步到 **GitHub**。

在本地电脑解压并使用 **VSCode** 打开项目根目录，在终端中执行：

```bash
# 1. 安装项目所有依赖
npm install

# 2. 启动前端 Vite 极速开发服务
npm run dev
```

启动完成后，在浏览器中打开命令行提示的本地地址（默认是 `http://localhost:3000`）即可游玩游戏。

---

## 2. 🖼️ 如何替换战机与敌机图片 (PNG)

我们在 Canvas 游戏渲染内核中已经集成了**智能自适应图片材质引擎**。你只需要在项目根目录中的 `public/assets/` 路径下放入对应的 PNG 图片，游戏启动时会自动检测并完成飞机的换肤！如果对应的图片文件不存在（比如当前默认情况），游戏会完美降级渲染现代科技感的发光矢量线框图。

### 图片命名规范与推荐分辨率

请在项目根目录下的 `public/` 目录中创建 `assets/` 文件夹（如果已经存在则无需创建），并将图片按以下文件名保存在 `public/assets/` 中：

1. **主角战机 (Player Ship):**
   - 文件路径: `public/assets/player.png`
   - 推荐尺寸: 64x64 或 128x128 像素
   - **重要提示**: 主角飞机图片头部应朝上（上方为前进开火方向）。
   
2. **普通敌机 (Basic Fighter):**
   - 文件路径: `public/assets/enemy_basic.png`
   - 推荐尺寸: 48x48 像素
   - **重要提示**: 敌机图片的头部应朝下（下方为俯冲进攻方向）。

3. **极速敌机 (Fast Interceptor):**
   - 文件路径: `public/assets/enemy_fast.png`
   - 推荐尺寸: 40x40 像素
   - 敌机图片头部应朝下。

4. **超重装战列舰敌机 (Heavy Cruiser):**
   - 文件路径: `public/assets/enemy_heavy.png`
   - 推荐尺寸: 96x96 像素
   - 敌机图片头部应朝下。

5. **行星级终极母舰 Boss (Boss Destroyer):**
   - 文件路径: `public/assets/enemy_boss.png`
   - 推荐尺寸: 160x160 像素
   - 敌机图片头部应朝下。

## 3. 🛡️ 智能兼容特性

- **矢量动画融合**: 即使你用 PNG 图片替换了主角或敌机的贴图，游戏引擎底层配备的**喷气式尾部等离子火焰**、**护盾吸收光圈**和**血条/核心光能球**仍会自动环绕在你的 PNG 图片周围，为换肤后的飞机附加一致的炫酷动效！
- **双模操控智能切换**: 游戏支持**键盘 (W/A/S/D/Space)** 与 **鼠标高精度悬停/拖拽** 双模，当你移动鼠标时，战机会以高帧率平滑跟随鼠标指针位置并启动自动射击，当按键盘方向键时键盘会立刻接管操控，带给你最极致的街机触感！
- **零配置零报错**: 离线状态或未放置 PNG 图片时，系统后台不会因为资源加载失败导致控制台报错或游戏黑屏，而是流畅无缝展示高对比度的霓虹矢量样式。
